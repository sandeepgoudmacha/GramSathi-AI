## 🎯 Creating Your First Training Program

### Step 1: Navigate to Training Tab
1. Click on **Group Management**
2. Look for the **Training** tab at the top of the page
3. You'll see the **Training Programs Management** section

### Step 2: Click "Create New Program" Button
- **Location**: Top-right corner of the Training section
- **Button**: **[+ Create New Program]** (highlighted in blue)
- This opens the **Create Training Program** modal form

### Step 3: Fill Out Program Details

#### 🎯 Program Title (Required)
- What is the training about?
- **Examples**: 
  - "Advanced Tailoring Course"
  - "Digital Payments & UPI Basics"
  - "Organic Food Processing"
  - "Mushroom Cultivation Techniques"

#### 📝 Description (Optional but Recommended)
- What will members learn?
- Topics covered
- Benefits for members
- **Example**: "Learn modern tailoring techniques, stitching patterns, use of sewing machines, and quality assurance. This course will help you start your own tailoring business."

#### 🏢 Provider/Trainer (Required)
- Who is providing/training this program?
- **Examples**: "NRLM Telangana", "State Agriculture Department", "Women's Federation", "NGO Partner"

#### ⏱️ Duration (Required)
- How many days?
- **Range**: 1-365 days
- **Common values**: 3, 5, 7, 14, 30 days

### Step 4: Set Schedule & Location

#### 📅 Start Date
- When does the program begin?
- Leave blank if flexible
- Format: **YYYY-MM-DD** (e.g., 2026-04-15)

#### 🏛️ Mode (How to Deliver)
Choose one:
- **🏪 Offline** - In-person training at a physical location
- **💻 Online** - Virtual training via video call, Zoom, Google Meet, etc.
- **🔄 Hybrid** - Mix of in-person and online sessions

#### 📍 Location (For Offline Programs)
- Full address or venue name
- **Examples**: 
  - "GramSathi Community Center, Nalgonda"
  - "SHG Office, Village Square"
  - "Government School Building, Hall No. 2"

### Step 5: Set Capacity

#### 👥 Max Participants
- How many members can enroll?
- **Minimum**: 1
- **Recommended**: 15-30 members per batch
- **Tip**: Once full, no more enrollments accepted. Plan accordingly!

### Step 6: Review and Create

- **Review** all information
- Click **[Create Program & Notify All Members]**
- System will:
  - Create the program
  - Send notifications to ALL {count} SHG members automatically
  - Members can now see and enroll in it

## 📊 Viewing Training Programs

### The Training Dashboard Shows:

#### 📈 Summary Cards (Top)
- **Total Programs**: How many active programs
- **Total Enrollments**: How many members across all programs
- **Total Capacity**: Total seats available across all programs

#### 📋 Programs List
Each program card shows:
- **Program Title** with description
- **Provider** and **Duration**
- **Mode**: (Offline/Online/Hybrid badge)
- **Enrollment Progress**: 
  - Current: "X / Y enrolled"
  - Percentage bar (color-coded):
    - 🟢 **Green** (0-50%) = Plenty of seats
    - 🟡 **Amber** (50-80%) = Getting full
    - 🔴 **Red** (80-100%) = Almost full
  - "X seats left"
- **Start Date** (if set)
- **Location** (if offline)

### Click a Program to See Enrollments

When you click on a program card, you'll see:

#### Program Overview
- Full program title
- Provider name
- Duration & Mode
- Location (if applicable)
- Start date (if set)
- Full description

#### Enrollment Statistics
- **Large number** = Total members enrolled
- **Progress bar** = Percentage full
- **Capacity** = Max participants

#### 📋 Enrolled Members Table
Shows all members who enrolled with:
- **👤 Member Name** - Full name of the member
- **📱 Phone** - Contact number
- **📍 Location** - Village/District
- **💼 Occupation** - Their profession
- **📅 Enrolled Date** - When they registered

## 🎯 Best Practices for Coordinators

### ✅ DO:
- **Plan ahead** - Decide training dates and topics before creating
- **Set realistic capacity** - Don't set too high if you can't deliver
- **Fill descriptions** - Help members understand what they'll learn
- **Choose right mode** - Match delivery method to topic and member needs
- **Check enrollments regularly** - Monitor who enrolled and plan accordingly
- **Communicate** - Contact members via phone for follow-up
- **Update members** - Let them know about materials, timing changes, etc.

### ❌ DON'T:
- **Create duplicate programs** - Check if similar training exists
- **Set very short notice** - Members need time to plan (aim for 1-2 weeks)
- **Overload capacity** - Quality over quantity for better outcomes
- **Create but forget** - Follow up with members for actual delivery
- **Change program details** - Notify members of any changes

## 🔍 Troubleshooting

### Problem: Can't find Training tab
**Solution**: 
- Make sure you're logged in as **Coordinator**
- Check you've opened **Group Management** page
- Training tab shows at the top with other tabs (Members, Loans, Savings, Training, Records)

### Problem: "Create New Program" button not visible
**Solution**:
- Refresh the page
- Check browser zoom level (try 100%)
- Try different browser (Chrome, Firefox, Edge)
- Clear browser cache

### Problem: Form not submitting
**Solution**:
- Ensure all **Required** fields are filled:
  - Program Title ✓
  - Provider ✓
  - Duration ✓
  - Max Participants ✓
- Check internet connection
- Wait for "Creating..." status to complete

### Problem: Members not seeing program
**Solution**:
- Wait a few seconds for notification to arrive
- Ask member to refresh their Skills & Training page
- Check member's notification panel for update
- Ensure program has `is_active = 1` in database

### Problem: Wrong enrollment numbers
**Solution**:
- Refresh the Training page (F5)
- Close and reopen browser
- Check database directly if persistent

## 📋 Coordinator Checklist

- [ ] Clear program title

- [ ] Realistic capacity based on your support capability
