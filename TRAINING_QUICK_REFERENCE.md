# 🎓 Coordinator Training Programs - Quick Reference

## ⚡ In 3 Steps

### Step 1️⃣ Open Training Tab
- Login → **Group Management** → **Training** tab

### Step 2️⃣ Create Program
- Click **[+ Create New Program]** button
- Fill program details (Title, Provider, Duration, Mode, Capacity)
- Click **[Create Program & Notify All Members]**

### Step 3️⃣ View Enrollments
- Programs appear in list with enrollment stats
- Click program → See all enrolled members with details

## 📋 Before Creating a Program

### Coordinator Checklist
- [ ] Do I have a trainer/provider confirmed?
- [ ] Are materials ready or sourced?
- [ ] Is venue available (if offline)?
- [ ] Have I planned the schedule?
- [ ] Is capacity realistic for what I can manage?
- [ ] Do I have at least 1-2 weeks lead time?
- [ ] Is the program title clear and descriptive?
- [ ] Have I identified target member group?

## 🚀 Tips for Success

### Do's ✅
1. **Create with enough lead time** (1-2 weeks)
2. **Fill description clearly** (what members learn)
3. **Choose appropriate capacity** (not too high)
4. **Update members** before start date
5. **Track who enrolled** for follow-up
6. **Provide clear venue details** for offline
7. **Send reminders** a day before

### Don'ts ❌
1. **Don't create last-minute** (members need notice)
2. **Don't set impossible capacity** (set realistic)
3. **Don't forget to follow up** (send materials)
4. **Don't change details** without notifying
5. **Don't create duplicate programs** (same topic)
6. **Don't set mode incorrectly** (mode ≠ delivery)

## ❓ FAQ

**Q: How do members see the programs?**
A: They go to Skills & Training tab → Training Programs section

**Q: Will members get notified?**
A: Yes! Automatic notifications sent to ALL active members when you create

**Q: Can I edit a program after creating?**
A: Currently no, but you can create a new one and deactivate old

**Q: What if the program is full?**
A: "Full" status shows, new members can't enroll, existing can see details

**Q: How do I contact enrolled members?**
A: Member phone numbers are visible on the enrollment details page

**Q: Can members see all programs?**
A: Yes, all active programs visible to all members
